# Coffee Sales Data Analysis

## Objective

Analyze coffee sales data to identify trends, customer behavior, and key business insights.

## Dataset

* Contains transaction-level sales data
* Includes product type, time of purchase, and revenue

## Tools Used

* Python
* Pandas
* Matplotlib / Seaborn

## Key Insights

* Identified top-selling coffee products
* Analyzed peak sales hours
* Observed customer purchasing patterns

## Project Structure

* data/ → dataset
* notebook/ → analysis code
* report/ → final report

## How to Run

1. Open the notebook in Jupyter Notebook
2. Run all cells
3. Ensure dataset path is correct
